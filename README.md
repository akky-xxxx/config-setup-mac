## how to install

1. get shell script
```shell
curl -O https://raw.githubusercontent.com/akky-xxxx/setup-mac/master/setting.sh
```

2. change permission
```shell
chmod 755 ./setting.sh
```

3. run shell script
```shell
sh ./setting.sh
```

4. install by ma

```shell
mas install app_id
```
