## how to install

1. sign in to app store

2. get shell script
```shell
curl -O https://raw.githubusercontent.com/akky-xxxx/setup-mac/master/setting.sh
```

3. change permission
```shell
chmod 755 ./setting.sh
```

4. run shell script
```shell
sh ./setting.sh
```
